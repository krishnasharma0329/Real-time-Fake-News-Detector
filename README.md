📰 Real-time Fake News Detector

An AI-powered web application that detects fake news in real-time using advanced Natural Language Processing (NLP) and Transformer models. This project leverages pre-trained models from Hugging Face to analyze news articles and social media content, providing instant credibility scores through an interactive Streamlit interface.

🚀 Features
🔍 Real-time fake news detection
🤖 Powered by Transformer-based NLP models
📊 Instant credibility score generation
🌐 User-friendly web interface using Streamlit
🧠 Uses Hugging Face pre-trained models
⚡ Fast and efficient text classification
🛠️ Tech Stack
Programming Language: Python
Libraries & Frameworks:
Hugging Face Transformers
NLP (Natural Language Processing)
Streamlit
